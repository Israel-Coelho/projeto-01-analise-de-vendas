# projeto-01-analise-de-vendas
Projeto de análise de dados focado em vendas, faturamento e desempenho comercial
## Objetivo
Este projeto foi desenvolvido com o objetivo de analisar dados de vendas de uma loja fictícia, buscando identificar padrões de faturamento, produtos de maior destaque, categorias mais relevantes e diferenças de desempenho entre regiões e meses.

## Base de dados
A base contém informações de pedidos realizados entre janeiro e abril de 2025, com as seguintes colunas:
- Data
- Pedido_ID
- Produto
- Categoria
- Regiao
- Quantidade
- Preco_Unitario
- Faturamento

## Perguntas de negócio
- Qual foi o faturamento total no período?
- Quais produtos venderam mais em quantidade?
- Quais produtos geraram mais faturamento?
- Qual categoria teve melhor desempenho?
- Qual região teve maior faturamento?
- Qual mês apresentou melhor resultado?

## Etapas realizadas
1. Organização da base de dados
2. Criação da coluna de faturamento
3. Construção de tabelas dinâmicas
4. Análise por produto, categoria, região e mês
5. Criação de gráficos para visualização dos resultados

## Principais insights
- O faturamento total do período foi de R$ 24.130.

## Conclusão
A análise mostrou que produtos de maior valor unitário tiveram impacto muito superior no faturamento total, com destaque para a categoria Computadores. Também foi possível identificar concentração de receita na região Sudeste e variações mensais de desempenho, o que pode apoiar decisões comerciais futuras.
